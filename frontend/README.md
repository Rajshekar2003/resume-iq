# AI Resume Analyzer — Frontend

React + Vite + Tailwind CSS frontend for the AI Resume Analyzer.

## Prerequisites

- Node.js 18+
- Backend running on `http://127.0.0.1:5000` (see `backend/README.md`)

## Install

```bash
cd frontend
npm install
```

## Run development server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

> **Important:** The backend Flask server must be running on port 5000 for API calls to work.
> From `backend/`: `python app.py`

## Build for production

```bash
npm run build
```
